## Quick Reference

### Configuration Files
- `~/.config/nvim/init.vim` - Main configuration file (Vimscript)
- `~/.config/nvim/init.lua` - Main configuration file (Lua alternative)
- `~/.config/nvim/lua/` - Directory for Lua modules

### Common Commands
- `:help` - Built-in help system
- `:checkhealth` - Diagnose Neovim setup
- `:PlugInstall` - Install plugins (vim-plug)
- `:TSUpdate` - Update TreeSitter parsers
